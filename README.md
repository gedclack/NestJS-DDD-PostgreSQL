## Description

+ Building API with NestJS implementing Domain-Driven Design
+ Use PostgreSQL with migration support

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## TypeORM Migration

```bash
# Generate changes
$ npx typeorm migration:generate -n <migration-name>
```
